<Update label="2024-11-15" description="v2.5.0" tags={["feature", "improvement"]}>

## New Features

- Added two new deal categories: `AI Tools` and `E-commerce Integrations`, making it easier for you to find specialized software.
- Introduced SHOP SUMO DAY scheduler in seller dashboard, allowing you to plan promotions automatically.

## Improvements

- Seller tools now load `>40%` faster with optimized caching.
- Enhanced search filters for buyers, supporting fuzzy matching on product descriptions.

## Bug Fixes

- Fixed intermittent checkout failures during high-traffic events.
- Resolved dashboard refresh issues on mobile devices.

  // New webhook for deal category updates
